# Basics
### BOPOMOFO~ Taiwanese Alphabet
How it works- 21 Initials (consonant), final (vowel), Head vowel
- inital + final + tone = a word
     - tones: ma (straight), ma' (rising), ma^ (down-up!), ma' (falling), ma*(sharp)
1a) B (bAA - Ex. Steamed bun)
2a) P (pAA - Ex. Grapes)
3a) M (mAA - Ex. Cat)
4a) F (fAA - Ex. Rice)
1b) D (dAA - Ex. Knife)
